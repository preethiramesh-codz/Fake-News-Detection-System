 Fake News Detection System

Project Overview

The Fake News Detection System is a Machine Learning-based web application that identifies whether a news article is Real or Fake.

 Features

* Fake News Detection
* Machine Learning Classification
* User-Friendly Streamlit Interface
* Real-Time Prediction
* Trained Using ISOT Fake and Real News Dataset

 Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Joblib

 Dataset

ISOT Fake and Real News Dataset

 How To Run

Install Dependencies


pip install -r requirements.txt


Run Application


streamlit run app.py


 Output

The system predicts whether the entered news content is REAL or FAKE.
